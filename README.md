# Smart File Organizer
پروژه مرتب کردن فایل های یک پوشه
## Goal
در این پروژه فایل های یک پوشه بررسی و سازماندهی می شوند.
## Description
هدف این است که کاربر یک پوشه را وارد کند برنامه پوشه را بررسی میکند و فایل ها را میخواند و برای هر فایل نوع ان را تشخیص میدهد و بررسی میکند ایا برای این فایل پوشه مقصد وجود دارد یا خیر اگر وجود نداشت ان را می سازد و فایل رو انتقال می دهد  در اخر امار را ذخیره می کند و گزارش
 (csv)تولید میکند
